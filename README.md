# Sorting Algorithms
## C# Bubble Sort
### Pseudocode
```
function main()
    int[] arr = { 7, 4, 5, 2}
    loop i from 0 to 3 //index 0-3
        loop j from 0 to 3 - i - 1
            if a[j] > a[j + 1] then
                tmp = a[j]
                a[j] = a[j + 1]
                a[j + 1] = tmp
            end If
        end loop
    end loop
end function
```
